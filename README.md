# orbital-tracker
A simple HTML + JavaScript + CesiumJS app that visualizes satellites and space debris in 3D using TLE data.

Features
.3D Earth using CesiumJS
.Renders satellite orbits from TLE
.Works entirely on the frontend
.No build tools, no backend
.Gemini button to know about the orbitals


Tech Stack
.HTML
.CSS
.JavaScript (vanilla)
.CesiumJS
